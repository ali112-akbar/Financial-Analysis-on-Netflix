# Financial-Analysis-on-Netflix
This project presents a full investment analysis of Netflix (NFLX) using a structured, institutional-style approach combining financial analysis, valuation modeling, and risk assessment.  The goal of this project is to evaluate whether Netflix is undervalued or overvalued based on fundamental analysis and to simulate real-world analyst workflows.
📊 Key Features
Financial statement analysis (Income Statement, Balance Sheet, Cash Flow)
Profitability metrics: EBIT Margin, Net Margin, ROIC
Capital structure analysis (Debt, Cash, Leverage ratios)
WACC calculation using CAPM
3-Scenario DCF Valuation (Bear / Base / Bull)
Sensitivity Analysis (WACC vs Terminal Growth)
Comparable Companies Analysis (Comps)
Monte Carlo Simulation (10,000 trials)
Tornado Analysis (key value drivers)
📈 Tools & Technologies
Python (Pandas, NumPy)
Data Visualization (Matplotlib, Seaborn)
yfinance (financial data extraction)
Financial Modeling techniques (DCF, Comps, Risk Analysis)
🎯 Key Insights
Netflix demonstrates strong revenue growth and margin expansion
ROIC significantly exceeds WACC, indicating value creation
Free Cash Flow is improving, reflecting a stronger business model
Valuation is highly sensitive to growth and discount rate assumptions
Current market pricing reflects high growth expectations
📌 Investment View

Netflix is a high-quality company with strong fundamentals, but valuation suggests a limited margin of safety under conservative assumptions.

👉 Recommendation: HOLD (with selective buying on market dips)
